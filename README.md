## Features

### Frontend Features
1. Eye-catching landing page with smooth animations using **Framer Motion**.  
2. Well-categorized company information for easy navigation.  
3. Easy contact system and mailing through the "Contact Us" form.  
4. Dynamic content updates via a custom **WYSIWYG editor** powered by **DraftJS**.  
5. PDF generation directly from website data stored in the database.  
6. Push page data (expedition updates) to **Facebook, Twitter, and WhatsApp**.  
7. Routing and navigation handled with **React Router**.  
8. Styled using **Styled-Components** and advanced CSS for responsive design.  

### Backend Features
1. Node.js and Express server with **RESTful API endpoints**.  
2. **MongoDB + Mongoose** for database management.  
3. Admin authentication.  
4. Environment-based configuration using `.env` for sensitive data like database URI and server port.

## Tech Stack

**Frontend:** HTML, CSS, React, Framer Motion, Styled-Components, DraftJS, React Router  
**Backend:** Node.js, Express.js, MongoDB, Mongoose  
**Other:** Git, Vercel (for deployment), PDF generation library, Mailing library, Social media API integration
